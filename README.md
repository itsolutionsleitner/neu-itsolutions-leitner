# IT Solutions Leitner – Website

Reines HTML/CSS/JS-Projekt, kein Framework, kein Build-Tool.

## Projektstruktur

```
itsolutions-leitner/
├── index.html
├── ueber-uns.html
├── leistungen.html
├── faq.html
├── kontakt.html
├── impressum.html
├── datenschutz.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
└── README.md
```

## Lokal testen

Projektordner in VS Code öffnen → `index.html` → Rechtsklick → "Open with Live Server".

## Deployment

1. `git add .`
2. `git commit -m "..."`
3. `git push`

Vercel deployt bei jedem Push auf `main` automatisch. Keine weiteren Schritte nötig.
